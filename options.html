Chrome flags required to enable Nano AI

Gemini Nano requires two Chrome flags to be enabled. Open each link below, set to Enabled, then relaunch Chrome.

  1. chrome://flags/#optimization-guide-on-device-model — Enables on-device model download
  2. chrome://flags/#prompt-api-for-gemini-nano-multimodal-input — Enables the Prompt API

After enabling both flags and relaunching, return here and select Nano AI.
